# Address Book Management System (C)

This is a console-based Address Book application developed in C.

## Features
- Add new contact
- Search contact by name, phone, or email
- Edit existing contact
- Delete contact
- List all contacts
- Save contacts to a CSV file

## Technologies Used
- C Programming
- File Handling
- Structures

## Files in Project
main.c – Main program  
addressbook.c – Functions for contact operations  
addressbook.h – Structure and function declarations  
database.csv – Stores contact data  

## How to Run

Compile the program:

gcc main.c addressbook.c

Run the program:

./a.out
